Cloud sync — use on any device

Optional. Connect a free Supabase project once; then log in on any device to access all your data.
One-time setup (≈2 minutes)
  1. Create a free project at supabase.com.
  2. Open SQL editor and run:
    create table vault (email text primary key, account jsonb, data jsonb, updated_at timestamptz); alter table vault enable row level security; create policy "rw" on vault for all using (true) with check (true);
  3. In Project settings → API, copy the Project URL and anon public key into the boxes above, then Connect & sync.
  4. On another device, log in with the same email/password — your data loads automatically.
